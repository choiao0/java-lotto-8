# java-lotto-precourse

### ⚙️ 기능 목록 정리

- [ ]  로또 구입 금액을 입력 받는다.
- [ ]  로또 발행 수량을 계산한다.
    - [ ]  금액이 1,000원으로 나누어 떨어지지 않는 경우 예외 처리한다.
- [ ]  수량만큼 로또를 발행한다.
    - [ ]  1개의 로또를 발행할 때 중복되지 않는 6개의 숫자를 뽑는다.
- [ ]  발행한 로또 수량 및 번호를 출력한다. 로또 번호는 오름차순으로 정렬하여 보여준다.
- [ ]  당첨 번호를 입력 받는다.
    - [ ]  번호는 쉼표(,)를 기준으로 구분한다.
- [ ]  보너스 번호를 입력 받는다.
- [ ]  사용자가 구매한 로또 번호와 당첨 번호를 비교한다.
    - [ ]  로또가 당첨되면 당첨 내역에 기록한다.
- [ ]  당첨 내역을 출력한다.
- [ ]  수익률을 계산한다.
- [ ]  수익률을 소수점 둘째 자리에서 반올림하여 출력한다.


### 🧩 예외 사항

사용자가 잘못된 값을 입력할 경우`IllegalArgumentException`을 발생시키고, "[ERROR]"로 시작하는 에러 메시지를 출력 후 그 부분부터 입력을 다시 받는다.

로또 구입 금액을 잘못 입력한 경우

- [ ]  금액이 1000 미만인 경우
- [ ]  금액에 숫자 외의 다른 문자가 있는 경우
- [ ]  금액이 1000으로 나누어 떨어지지 않는 경우

당첨 번호를 잘못 입력한 경우

- [ ]  당첨 번호가 1~45 범위 밖인 경우
- [ ]  당첨 번호가 6개가 아닌 경우
- [ ]  당첨 번호에 숫자와 쉼표 외의 다른 문자가 있는 경우
- [ ]  당첨 번호에 중복된 숫자가 있는 경우

보너스 번호를 잘못 입력한 경우

- [ ]  보너스 번호가 1~45 범위 밖인 경우
- [ ]  보너스 번호가 1개가 아닌 경우
- [ ]  보너스 번호에 숫자 외의 다른 문자가 있는 경우
